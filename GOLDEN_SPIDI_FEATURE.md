# 🌟 Золотой Спиди - Кнопка Клика

## Как разблокировать

Золотой Спиди становится доступен как кнопка клика двумя способами:

### 1. Медаль 100K 🏅
- При достижении **100 000 кликов**
- Автоматически разблокируется `hasGoldenSpidi = true`
- Уведомление: "🏅 Медаль 100K + Золотой Спиди разблокирован!"

### 2. Подарок День 5 🎁
- При получении подарка **5-го дня** (Золотой Спиди)
- Даёт +50 к силе клика навсегда
- Разблокирует Золотого Спиди как кнопку
- Уведомление: "🌟 Золотой Спиди! +50 + кнопка!"

## Как переключить кнопку

После разблокировки в разделе **Настройки** появляется новый раздел:

### 🎯 Кнопка клика

Два варианта:
1. **Обычная** - стандартная кнопка (по умолчанию)
2. **Золотой Спиди** - золотая кнопка

### Как выглядит:
- **ПК/Планшет**: 2 кнопки рядом с превью (64x64px)
- **Телефон**: 2 кнопки рядом с превью (56x56px)
- Активная кнопка подсвечивается синим (обычная) или оранжевым (золотой)
- Показывается галочка ✓ под активной кнопкой

## Техническая реализация

### GameState
```typescript
hasGoldenSpidi: boolean          // Разблокирован ли золотой спиди
selectedClickButton: 'default' | 'golden_spidi'  // Текущая кнопка
```

### Константы
```typescript
export const CLICK_BUTTON_IMAGES = {
  default: 'https://i.ibb.co/GQ1C2zFz/df81b79e-82c7-4e35-b2cd-8a69abe58eb5.jpg',
  golden_spidi: 'https://i.ibb.co/gppjW5R/1775898076670.jpg',
}
```

### Автосохранение
- Выбор кнопки автоматически сохраняется в localStorage
- При перезагрузке восстанавливается последний выбор
- Если `hasGoldenSpidi = false`, раздел в настройках скрыт

## Примечания
- ✅ Работает на ПК, планшетах и телефонах
- ✅ Плавная анимация переключения
- ✅ Сохраняется между сессиями
- ✅ Доступно только после разблокировки
