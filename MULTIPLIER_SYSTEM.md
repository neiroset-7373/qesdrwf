# 🔥 Система множителей (24 часа)

## Как работает

Множители **ЗАМЕНЯЮТ** силу клика, а не умножают её.

### Пример:
- У вас сила клика = 1
- Покупаете **x3 множитель** за 450 монет
- **Сила клика становится = 3** (не 1×3, а просто 3)
- Каждый клик даёт 3 монеты
- Через 24 часа множитель истекает
- **Сила клика возвращается к базовой** (1 + постоянные бонусы)

## Правила

### ✅ Можно:
- Купить один множитель на 24 часа
- Использовать множитель + постоянные бонусы (Золотой Спиди)

### ❌ Нельзя:
- Купить второй множитель пока первый активен
- Уведомление: "⚠️ Другой множитель активен!"
- Купить множитель повторно (максимум 1 раз)

## Постоянные бонусы (не истекают)

### Золотой Спиди (+50 к базовой силе клика)
Получается:
- При 100 000 кликов (Медаль 100K)
- Или при получении подарка День 5

**Как работает с множителями:**
- Базовая сила = 1
- Получаете Золотого Спиди → Базовая сила = 51
- Покупаете x3 → Сила клика = 3 (не 51×3!)
- Множитель истекает → Сила клика = 51 (возврат к базе)

## Истечение множителя

### Когда истекает:
- Автоматически через 24 часа после покупки
- Уведомление: "⏰ Множитель истёк, сила клика вернулась к 51"

### Что происходит:
1. `activeMultiplier` устанавливается в `null`
2. `clickPower` возвращается к `baseClickPower`
3. Можно купить новый множитель

## Техническая реализация

### GameState
```typescript
clickPower: number        // Текущая сила клика
baseClickPower: number    // Базовая сила (1 + постоянные бонусы)
activeMultiplier: {
  id: string
  multiplier: number
  expiresAt: string       // ISO timestamp
} | null
```

### Логика покупки множителя
```typescript
setState(s => ({
  ...s,
  coins: s.coins - mult.cost,
  clickPower: mult.multiplierValue,  // Заменяем силу клика
  activeMultiplier: { ... }
}))
```

### Логика истечения
```typescript
setState(s => ({ 
  ...s, 
  activeMultiplier: null,
  clickPower: s.baseClickPower  // Возврат к базе
}))
```

### Логика клика
```typescript
// Просто используем clickPower
const newCoins = s.coins + s.clickPower
```

## Примеры сценариев

### Сценарий 1: Начало игры
1. Базовая сила = 1
2. Покупаете x5 → Клик = 5
3. Через 24ч → Клик = 1

### Сценарий 2: С Золотым Спиди
1. Получаете Золотого Спиди → Базовая = 51, Клик = 51
2. Покупаете x100 → Клик = 100
3. Через 24ч → Клик = 51

### Сценарий 3: Получение Золотого Спиди во время активного множителя
1. Покупаете x10 → Клик = 10
2. Получаете Золотого Спиди → Базовая = 51, Клик остаётся 10
3. Через 24ч → Клик = 51
