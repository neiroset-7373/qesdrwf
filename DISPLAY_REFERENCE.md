# 📊 Справка по отображению силы клика

## Где отображается сила клика

### 1. **Шапка игры (Header)**

#### ПК/Планшет:
```
[Иконка силы] 1
Сила клика
```

#### Телефон:
```
[Иконка] 1
Сила клика
```

**Формат:** `{state.clickPower}` (без "+")

---

### 2. **Статистика в игре (Stats bar)**

#### ПК/Планшет:
```
Сила клика
1
```

#### Телефон:
```
[Иконка] 1
Сила клика
```

**Формат:** `{state.clickPower}` (без "+")

---

### 3. **Индикатор активного множителя**

#### ПК/Планшет:
```
🔥 Множитель x3 активен · Сила клика = 3
Истекает: 15 янв., 14:30
```

#### Телефон:
```
🔥 x3 · Клик = 3
До: 15 янв., 14:30
```

**Формат:** `Сила клика = {state.clickPower}` (без "+")

---

### 4. **Уведомления**

#### При покупке множителя:
- ПК: `✅ Множитель x3 активен! Сила клика = 3`
- Телефон: `✅ Сила клика = 3!`

#### При истечении множителя:
- ПК: `⏰ Множитель истёк, сила клика вернулась к 51`
- Телефон: `⏰ Истёк, клик = 51`

#### При получении Золотого Спиди:
- `🌟 Золотой Спиди! +50 клик + разблокирована кнопка!`
- (здесь "+" означает прибавление, а не отображение текущего значения)

---

## Анимация при клике

### Всплывающий текст над кнопкой:
```
+3  (белый текст, всплывает вверх)
```

**Формат:** `+{value}` где value = текущая сила клика

Это анимация показывает сколько монет добавилось за клик, поэтому здесь "+" корректен.

---

## Правила отображения

### ✅ БЕЗ плюса (+):
- Шапка игры
- Статистика
- Индикатор множителя
- Уведомления о текущем значении

### ✅ С плюсом (+):
- Анимация при клике ("+3")
- Уведомления о получении бонусов ("+50 клик")
- Подарки с монетами ("+1000 монет")

---

## Примеры

### Сценарий 1: Обычная игра
```
Шапка: 1
Клик: +1 (анимация)
Уведомление: "Сила клика = 1"
```

### Сценарий 2: С множителем x10
```
Шапка: 10
Клик: +10 (анимация)
Индикатор: "🔥 Множитель x10 активен · Сила клика = 10"
```

### Сценарий 3: С Золотым Спиди + множитель
```
Шапка: 100
Клик: +100 (анимация)
Индикатор: "🔥 Множитель x100 активен · Сила клика = 100"
После истечения: "⏰ Множитель истёк, сила клика вернулась к 51"
Новая шапка: 51
```
